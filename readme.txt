This is a Task manager app built on nodejs, using express framework and
Angular 2. it was built following a tutorial by traversymedia.com on 
"How to build a MEAN app from scratch". Angular 2 is a dependency and you need to 
find documentation for it using web archives such as web.archive.org to get
the relevant installation documentation. This link: https://web.archive.org/web/20161011223739/https://angular.io/docs/ts/latest/quickstart.html
Has the correct version of Angular for the purposes of the tutorial found here: https://www.youtube.com/watch?v=PFP0oXNNveg
